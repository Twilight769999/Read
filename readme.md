\### 我的读书规则

1\. 一本书只写一页核心笔记（A4 正反面原则）

2\. 每本书必须画一张手绘思维导图（存在对应文件夹）

3\. 读完立刻写 200 字以内读后感，不拖延

4\. 每年 12 月 31 日写《年度 Top 5》
